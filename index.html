About TinkerUSB

Tinker USB is the World's First WiFi based BadUSB. This lets you control any device over WiFi, while all you have to do is connect it to the device and it begins executing ducky script

Tinker USB is based on Open Source hardware, so you can reprogram it to different applications or you can choose from the wide range of scripts that you can run out of the box. With the Tinker USB, you can do a lot more than what you think you can.

The Salient features of TinkerUSB are :

  • Control Devices using WIFI.
  • Executes Automated Scripts.
  • Blocks Intruder in your network.
  • Share Passwords securely with others.
  • Share Media.

Quick Start

Step 1

Plug the TinkerUSB Pro/Mini Version to the device you want to control

Step 2

Tinker USb broadcasts a WiFi by itself. Connect to the Wifi with following Credentials.
SSID : TinkerUSB
Password : TinkerUSB@123

Step 3

Once Connected, Open Any browser of your choice. Enter 192.168.4.1 in the URL search Bar.

Frequently Asked Questions

  • How do I start using the tinker USB?

    To start using the tinker USB all you have to do is plug it into your computer, the tinker USB will automatically take you to our getting started page.

  • How do I load new scripts and execute it onto the on board storage?

    you can connect to the hosted network and upload and manage your scripts.

  • How much on board storage does the Tinker USB have?

    The board comes with around 3mb of storage space, this is a lot considering the most complex of ducky scripts only occupy hundreds of bytes of storage memory.

  • How do I reprogram the Tinker USB?

    You can download the Arduino IDE and use that to upload scripts to the board. Full instructions will be provided in the website.

  • Does the tinker USB support my smart device?

    The tinker USB works on most devices with a USB input, so it supports windows, Linux and Mac OS. It also works with most smart TVs and the PS4 and Xbox One, it runs on android and we are working on an iPhone application which should come out soon.
